<objective>
Apply `@moduledoc tags: [:stable]` or `tags: [:adapter]` to every public Rendro module per the D-04/D-05 tier lists, and update `mix.exs` to: (1) reconcile `groups_for_modules` with the corrected surface, (2) inject badge CSS and JS via `before_closing_head_tag`.

Output: All public modules tagged; mix.exs updated with badge injection and corrected groups; `mix docs` produces colored Stable/Adapter badges.
