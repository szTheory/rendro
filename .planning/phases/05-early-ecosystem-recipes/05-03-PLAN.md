Today the adapter:
- Silently fabricates an empty `%Swoosh.Email{}` and returns it for `Mailglass.*` wrapper shapes that have neither a `:swoosh` nor an `:email` field (CR-01 — silent data loss).

Purpose: Restore the documented `{:error, %Rendro.Error{}}` failure contract and surface unrecognized wrapper shapes via a typed error tuple so callers can react instead of being lied to or crashed.
