## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| stage stop_meta → telemetry handlers | Handlers (Threadline, future custom) consume the metadata map. New `:error` key must not leak unsanitized data. |
| `Rendro.Error.from_stage/3` → telemetry handlers | The `error.reason` atom flows into `meta.error.kind`. Atoms are bounded; no string interpolation. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-06-01 | Information Disclosure | `stage_stop_meta/5` `:error` map | mitigate | The map is `%{kind: error.reason, stage: error.stage}` — both atoms, no `error.why` (which can contain free-form strings) and no `details` (which can hold arbitrary terms). Verified by acceptance criteria asserting only `:kind` and `:stage` keys. |
| T-06-02 | Denial of Service | `derive_page_count/2` helper | accept | Pattern-matches on `%Rendro.Document{pages: pages}` and calls `length/1`. Lists can be unbounded but pages list size is the operator's input — same risk surface as the existing `length(doc.pages)` calls; no new attack surface. |
| T-06-03 | Tampering | `Rendro.Error.from_stage/3` for `:validate` reasons | mitigate | New `next_step` clauses are static strings — no string interpolation of caller-provided `reason`. |
| T-06-04 | Repudiation | Telemetry handlers see `:error` event | accept | This phase strengthens auditability (preserves `page_count` on error). No regression. |
</threat_model>

      defp span(stage, base_meta, fun, last_doc) do
        start_meta = Map.put(base_meta, :stage, stage)

        :telemetry.span([:rendro, :pipeline, stage], start_meta, fn ->
          case fun.() do
            {:ok, result} ->
              {{:ok, result}, stage_stop_meta(stage, :ok, result, last_doc, base_meta)}

            {:error, %Error{} = error} ->
              {{:error, error}, stage_stop_meta(stage, {:error, error}, nil, last_doc, base_meta)}

              {{:error, error}, stage_stop_meta(stage, {:error, error}, nil, last_doc, base_meta)}
          end
        end)
      end

      defp stage_stop_meta(stage, status_or_error, result, last_doc, base_meta) do
        page_count = derive_page_count(result, last_doc)

        base = %{

          document_type: base_meta.document_type,
          deterministic: base_meta.deterministic,
          stage: stage,
          status: if(status_or_error == :ok, do: :ok, else: :error),
          page_count: page_count,
          byte_size: byte_size

        case status_or_error do
          :ok -> base
          {:error, %Error{} = e} -> Map.put(base, :error, %{kind: e.reason, stage: e.stage})
        end
