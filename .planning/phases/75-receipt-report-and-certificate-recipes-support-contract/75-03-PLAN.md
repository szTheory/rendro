## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Certificate.document/2 caller input | Caller-supplied data map; validate_data!/1 is the trust boundary |
| data.brand | Caller-supplied brand map; validate_brand!/1 called before any asset registration — prevents partial registration state |
| Certificate body text | Caller controls data.body string length; no public text-height measurement API means very long strings may overflow to multiple pages (known limitation documented in RESEARCH.md Pitfall 7) |
