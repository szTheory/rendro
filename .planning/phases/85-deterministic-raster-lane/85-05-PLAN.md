<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| pdfium-cli binary -> rendered PNG bytes | The PNG reference is trusted only if generated by the version+sha256-pinned binary from `priv/pdfium_pin.json`. |
| CI/container bless command -> committed refs | Blessing mutates `priv/raster_refs/`; it must remain restricted to the pinned environment. |
| committed refs -> support matrix claims | `byte_deterministic_on_pinned_container: supported` is truthful only if the matrix points at a committed digest checked by tests. |

| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-85-SC | Tampering | Package or binary installation | mitigate | No new packages. The only binary is the existing pdfium-cli lane binary, pinned by version and sha256 in `priv/pdfium_pin.json`. |
