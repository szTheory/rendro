<objective>
Ship `Rendro.Format` (D-11): a small PURE, deterministic, locale-free formatting module the Statement recipe uses to format BOTH caller-supplied amounts AND recipe-computed running/carried balances.

Purpose: The recipe must format the Decimal balances it computes itself, so pre-formatted-strings-only is impossible; and putting CLDR/gettext in core would make output runtime-locale-dependent (non-deterministic), violating the byte-identical determinism contract. A pure default + `:formatters`/`:labels` escape hatch is the locked answer.

Output: `lib/rendro/format.ex` with deterministic default money/date formatting and default labels; full unit tests including a determinism assertion. NO CLDR, gettext, or any locale-aware dependency.
</objective>
