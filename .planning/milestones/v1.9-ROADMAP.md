# Milestone v1.9: Embedded Artifact Surfaces

**Status:** Complete (2026-05-06)
**Phases:** 48-50
**Total Planned Phases:** 3

## Overview

`v1.9` extends Rendro's deterministic authored surface with document-level embedded files and curated link annotations, while explicitly deferring encryption, signatures, and broad annotation workflows. The milestone keeps the public contract narrow and proof-backed, matching Rendro's current product DNA.

## Phases

### Phase 48: Embedded File Core Surface (Complete)

**Goal**: Add a deterministic authored boundary for document-level embedded files and serialize it into the PDF catalog/object graph without widening into page-level review features.
**Depends on**: Phase 47
**Planned focus:**

- Embedded-file domain model and public API
- Deterministic metadata validation
- Catalog names-tree and embedded-file serialization

### Phase 49: Curated Link Annotation Surface (Complete)

**Goal**: Add the narrowest useful non-form annotation surface through deterministic external-URI and internal-destination links.
**Depends on**: Phase 48
**Planned focus:**

- Link annotation authored API
- Destination and URI validation
- Reuse of existing `/Annots` writer seam without exposing a generic annotation escape hatch

### Phase 50: Support-Boundary and Proof Closure (Complete)

**Goal**: Close the milestone with truthful docs, support-matrix updates, and proof artifacts for the embedded-artifact surface.
**Depends on**: Phase 48, Phase 49
**Planned focus:**

- `priv/support_matrix.json` contract updates
- Docs-contract coverage and guide updates
- Structural proof and any narrow viewer-proof claims actually backed by evidence

## Milestone Summary

**Key Decisions:**

- Split embedded artifact surfaces from encryption and signatures instead of keeping them in one milestone.
- Treat document-level embedded files as the primary next surface; not page-level file-attachment comments.
- Limit annotation scope to links only in `v1.9`.

**Deferred Beyond v1.9:**

- Native PDF encryption
- Signature fields and external signing preparation
- Generic annotations, comments, and review workflows
